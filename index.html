coding-intern

A procedural memory system for small language models. Captures every interaction as an experience, clusters similar patterns, and consolidates them into reusable skills — so models actually learn and improve over time.

how it works

Capture. Cluster. Consolidate. Improve.

01

Every interaction is an experience

Each task → execution → result is recorded with learning signals: novelty, outcome value, and frequency.

02

Similar experiences cluster together

Clusters of 3+ related experiences become skill candidates for the next consolidation cycle.

03

Clusters become reusable skills

Each cluster generates a prompt template, trigger keywords, and verification checks — a new SkillNode.

04

The router improves automatically

New skills are indexed in FAISS. Over time, more tasks are handled locally without fallback.

architecture

Designed for minimal hardware.

Runs on CPU-only machines, Apple Silicon, and NVIDIA GPUs.
